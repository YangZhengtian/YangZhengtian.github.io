最美linux！

elementaryOS loki

​ 假期实习时为了go开发，体验了一下linux操作系统，然而只有elementary让我愿意一遍一遍的重装调试，原因只有一个，这个系统太美了，见过其界面后其他的ubuntu系统根本不忍直视。有时候觉得颜控也是一种强迫症@_@。

​ 自带的壁纸更是让我无法自拔。以下是我在实际使用时从刚装系统开始的一些命令和操作方法。

  • 输入法安装：

sudo apt update

sudo apt install fcitx-googlepinyin

键盘布局只留English, language改Chinese

  • 添加repository命令：

sudo apt-get install python-software-properties

sudo apt-get install software-properties-common

  • 更新Vim

sudo apt-get remove vim-common

sudo apt-get install vim

  • error：E: 无法获得锁 /var/lib/dpkg/lock -open(11:资源暂时不可用)

sudo rm /var/cache/apt/archives/lock

sudo rm /var/lib/dpkg/lock

  • 启动应用所需库

sudo apt-get install libgconf2-4

  • 安装WeChat

sudo apt install npm

tar -xzf wechat.tar.gz

npm install && npm start

  • Typora

sudo apt-key adv –keyserver keyserver.ubuntu.com –recv-keys BA300B7755AFCFAE

sudo add-apt-repository ‘deb https://typora.io ./linux/‘

sudo apt-get update

sudo apt-get install typora

  • Postman & Gogland

tar -xzf …

  • go环境

sudo tar -xzf … -C /usr/local

cd etc

sudo vi profile

  • 创建桌面启动器

cd /usr/share/applications

sudo vim xxx.desktop

目前只用到这些，随时更…
