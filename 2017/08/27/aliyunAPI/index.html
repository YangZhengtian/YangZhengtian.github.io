本文以新闻头条API为例介绍

在阿里云API市场可以找到很多实用的API，有一些免费的可供我们练习使用，阿里云市场里的产品给了curl、Java、C#、PHP、Python、ObjectC这几种语言的示例，我参照其Java和Python的示例用go语言实现了API的使用。

首先来看一下Java和Python的示例：

从两段代码中可以看到前面定义了一些连接需要的常量，之后调用了请求api的方法，当然go语言里也有相应的方法封装在go语言的库里，只是我们第一次使用时不知道，这时可以去golang官网的package里查询相应的方法，并且看其源码得知其使用方法。

在appcode那里，因为go语言包里给的Header.Add函数只有两个参数，所以要注意不能用python中的三参数方法，要将APPCODE加在常量前并空一格。

在最后需要用json.Unmarshal解析响应数据，传给前端标准的Json格式。

package newsHeadlines

import (
    "encoding/json"
    "io/ioutil"
    "log"
    "net/http"

    "github.com/labstack/echo"
)

const (
    host     = "http://toutiao-ali.juheapi.com"
    path     = "/toutiao/index"
    method     = "GET"
    appcode     = "APPCODE e84e5f0a9341480fb16f2cfc8092a47b"
    querys     = "type="
    Succeed     = 0
)

type newsType struct {
    NewsType string `json:"type"`
}

func GetNews(c echo.Context) error {
    var (
        err       error
        data      map[string]interface{}
        url       string
        inputType newsType
    )

    if err = c.Bind(&inputType); err != nil {
        log.Println("Input error")

        return err
    }

    client := &http.Client{
        CheckRedirect: nil,
    }

    url = host + path + "?" + querys + inputType.NewsType
    req, err := http.NewRequest(method, url, nil)
    if err != nil {
        log.Println("Request error")

        return err
    }

    req.Header.Add("Authorization", appcode)
    resp, err := client.Do(req)
    if err != nil {
        log.Println("Client error")

        return err
    }

    news, err := ioutil.ReadAll(resp.Body)
    if err != nil {
        log.Println("Read error")

        return err
    }

    err = json.Unmarshal(news, &data)
    if err != nil {
        log.Println("Unmarshal error")

        return err
    }

    return c.JSON(Succeed, &data)
}

代码采用echo框架方便一些基础操作的实现，阿里云给的debug调试平台可以对我们的调试起到很大的帮助。
